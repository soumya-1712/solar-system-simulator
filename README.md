# Solar System Simulator

An interactive 3D Solar System Simulator built with React, Three.js, and Vite.

## Features

- **Real-time 3D Rendering**: High-quality visualization of the Sun, planets, and moons.
- **Interactive Exploration**: Zoom, pan, and rotate to explore the solar system.
- **Detailed Information**: Click on any planet to see facts like mass, diameter, and more.
- **Simulation Controls**: Pause/Play and adjust the speed of the simulation.
- **Visual Effects**: Sun bloom, realistic textures, and starfield background.

## Installation

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Building for Production

To create a production build:

```bash
npm run build
```

The built files will be in the `dist` directory.

## Tech Stack

- **React**
- **Three.js**
- **@react-three/fiber**
- **@react-three/drei**
- **Zustand** (State Management)
